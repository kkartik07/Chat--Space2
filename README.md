# Installation
## Running locally
### Make sure you have NodeJS and npm installed.
  #### 1. To download the code, you can either clone this repo or manually download the codes zip file.
  ##### For cloning
  ````
  git clone https://github.com/kkartik07/Chat--Space2.git
  cd chat--space
  ````
#### 2. Then install the dependencies using :
   ````
   npm install 
   ````
#### 3. Next, npm start to open the code on localhost:

   ````
   npm start
   ````
